A professional and interactive news analytics dashboard** powered by AI and RSS feeds. Analyze sentiment, detect bias, extract key topics, and visualize trends across multiple news sources.

## Features

- Real-time news aggregation from RSS feeds like CNN, BBC, and Google News.  
- Sentiment analysis using VADER and TextBlob for each news article.  
- Bias detection for evaluating subjectivity in news summaries.  
- Topic extraction using TF-IDF for key topics in news articles.  
- **Interactive charts**:  
  - Pie chart for sentiment distribution  
  - Line chart for sentiment trends over time  
  - Bar chart for country-wise sentiment  
  - Top keywords chart  
- Filter options in sidebar: filter by sentiment (Positive, Neutral, Negative).  
- Curated example news links in sidebar for quick reference.  
- Professional UI with custom styling and interactive summary metrics.
